# API Documentation

## À venir

Cette section contiendra la documentation de l'API REST pour le projet E-commerce Dashboard.

## Endpoints prévus

### Users

- `GET /api/users` - Liste des utilisateurs
- `GET /api/users/{id}` - Détails d'un utilisateur

### Transactions

- `GET /api/transactions` - Liste des transactions
- `GET /api/transactions/{id}` - Détails d'une transaction

### A/B Tests

- `GET /api/ab-tests` - Liste des tests A/B
- `POST /api/ab-tests` - Créer un nouveau test
- `GET /api/ab-tests/{id}/results` - Résultats d'un test

### KPIs

- `GET /api/kpis` - Tous les KPIs
- `GET /api/kpis/{metric}` - KPI spécifique
